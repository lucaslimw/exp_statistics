
# Estatística Experimental - Listas de Exercícios

Este repositório contém os códigos desenvolvidos para a resolução das listas de exercícios da disciplina **Estatística Experimental**, cursada durante o doutorado.

Os scripts foram elaborados como parte das atividades da disciplina e têm como objetivo aplicar, na prática, os métodos estatísticos discutidos em aula.

---

## 📂 Estrutura do repositório

O repositório está organizado de acordo com as listas de exercícios:
├── lista_1
├── lista_3
├── lista_4
├── lista_5
├── lista_6
├── lista_7
├── lista_8
├── lista_9
└── README.md


Cada pasta contém os códigos utilizados na resolução dos exercícios correspondentes.

---

## Linguagens e ferramentas

As análises foram desenvolvidas utilizando principalmente:

- **R**
- **Python** com **Jupyter Notebook**

As bibliotecas e pacotes utilizados variam conforme os exercícios de cada lista.

---

## Objetivo

Este repositório tem como finalidade:

- Organizar os códigos utilizados nas listas da disciplina
- Servir como material de consulta e estudo
- Registrar a aplicação computacional dos métodos de **estatística experimental**

---

## ⚠️ Observação

Os códigos foram desenvolvidos para fins acadêmicos e podem não representar implementações totalmente otimizadas.
